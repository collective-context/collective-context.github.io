## Du arbeitest mit AI-Tools und willst produktiver werden?

Dann bist du hier richtig. Collective Context dokumentiert Werkzeuge
und Workflows für alle, die AI-Agenten professionell einsetzen —
vom ersten `curl` bis zum Multi-Agent-Orchester.

---

## 🏢 Webagenturen & Freelancer

**Dein Szenario:** Du baust Websites für Kunden. Bisher: manuell coden
oder Website-Builder klicken. Jetzt: Claude Code generiert die Site,
GitHub Sync deployed sie automatisch auf den Server.

**Was du brauchst:**
- Claude Code (Terminal-Agent) → [Schritt 1](/setup/claude-code/)
- XED /TUI (Session-Browser) → [Schritt 2](/setup/xed-tui/)
- Zed Editor (optional) → [Schritt 3](/quickstart/setup/)
- Hosting mit GitHub Sync → [Hosting@ /Pakete](https://cbuzz.io/hosting/)

**Zeitaufwand:** Wenige Minuten Setup, nach der Lernkurve super produktiv.

veröffentlicht sie automatisch via Admin API.

## Was alle gemeinsam haben

- **Terminal** — du arbeitest in einem Terminal (Linux, macOS, Windows WSL)
- **Claude Code** — Anthropics AI-Agent im Abo Modell anstatt Token-Verrechnung
- **XED /TUI Dirigent**  — Anstatt Blackbox, die KI transparent dirigieren
- **Lokale Dateien** — alles bleibt auf deinem Rechner, kein Cloud-Lock-in
- **Open Source** — MIT/Apache 2.0, kein proprietärer Vendor
