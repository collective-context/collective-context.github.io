## 🚀 Quick Start in 5 Minuten

```bash
# 1. Install CCC
pip install ccc

# 2. Setup tmux layout
ccc tmux init --agents 4

# 3. Start orchestration
ccc orchestrate --start all

# 4. Sync context
ccc context init "Your project"
```

### Sequential Pipeline
```
Claude-1 → Aider-1 → Claude-2 → Aider-2 → Production
```

```
[Claude-1 + Claude-2] ⟷ [Aider-1 + Aider-2]

```
