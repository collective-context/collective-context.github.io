## Der Workflow — in 60 Sekunden erklärt

```
Du gibst Claude Code einen Auftrag im Terminal:
  "Erstelle eine Landing Page mit Hero, Features und Footer."

Claude Code schreibt die Dateien und committed sie in GitHub.

GitHub Sync deployed automatisch auf deinen Server.

Fertig. Deine Website ist live.
```

Alles was du brauchst: **Claude Code** (Schritt 1) + **XED /TUI** (Schritt 2)
um deine Sessions zu organisieren. Optional: **ZED Editor** als IDE.

→ [Setup starten — Schritt 1: Claude Code](/setup/claude-code/)

## Was ist Collective Context?

Collective Context dokumentiert wie AI-Agenten im Terminal zusammenarbeiten —
mit geteiltem Projektgedächtnis (`CLAUDE.md`), offenen Standards (`ACP`),
und ohne proprietären Lock-in.

**Für Einsteiger:** [Setup (4 min)](/setup/claude-code/) →
installiere Claude Code und XED /TUI, starte sofort.

**Für Fortgeschrittene:** [Konzepte](/zed/acp/) →
ACP, Dual-Agent Patterns, Multi-Agent-Orchestrierung.

**Open Source** · EU-Hosting kompatibel · DSGVO-konform
